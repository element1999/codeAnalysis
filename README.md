# CodeMind

AI-powered code understanding and documentation tool.

## Features

- **Automatic Documentation**: Analyze codebase and generate structured Wiki documentation
- **Smart Q&A**: Answer code-related questions based on context (RAG architecture)
- **Code Graph**: Build dependency relationships between functions/classes/modules
- **Incremental Updates**: Support partial updates after code changes

## Installation

```bash
pip install -e .
```

## Usage

### Initialize project
```bash
codemind init
```

### Build documentation
```bash
codemind build
```

### Start chat
```bash
codemind chat
```

### Check status
```bash
codemind status
```

### Clean cache
```bash
codemind clean
```

## Configuration

Config file: `.codemind/config.json`
